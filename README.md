# canary_chat
一个仿微信的H5+应用，使用mui组件库来构建接近原生APP的应用。后端采用springboot + mybatis，使用netty搭建websocket服务器用于聊天功能。  
## 技术栈
 - HTML/CSS/JavaScript
 - mui组件库
 - H5+ API
 
## 服务端
